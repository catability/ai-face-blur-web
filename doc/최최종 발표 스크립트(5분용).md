# 2 페이지: 문제 정의 및 배경
CCTV나 인터넷 영상은 한번 공유되면 영구 삭제가 어려워, 심각한 사생활 침해를 일으킵니다.

그래서 수술실, CCTV 영상이나 개인 콘텐츠 등은 배포 전 모자이크 처리가 필요한데요.

하지만 이 작업을 수작업으로 일일이 처리하는 것은 엄청난 시간과 노력이 낭비됩니다.

저희는 이 필수적이지만 번거로운 문제를 해결하기 위해, AI 자동화 솔루션이 필요하다고 판단했습니다.

# 5 페이지: 주요 기능 및 기술 스택
네, 방금 시연에서 보신 기능들은 다음과 같은 기술 스택으로 구현했습니다.
