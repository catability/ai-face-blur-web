# ai-face-blur-web
AI 활용 얼굴 인식 자동 블러 처리 웹서비스 구현

---

2025년도 정보보안학과 졸업 작품 프로젝트입니다.

AI를 기반으로 영상에 등장하는 얼굴을 자동으로 탐지하고 블러 처리를 진행해 손쉽게 개인정보를 보호할 수 있는 웹 서비스를 구축하려고 했습니다.

일부 버그나 코드 정리는 되어 있지 않습니다...

---


작품명: AI 기반 영상 얼굴 자동 블러 처리 웹서비스 구현

작품설명: <br>
  저희 작품은 AI 딥러닝 기술(YOLO)을 활용해 영상 속 얼굴을 자동으로 탐지하고 블러 처리하는 웹 서비스입니다.<br>
사용자가 영상을 업로드하면 AI가 실시간으로 얼굴을 인식해 익명화를 수행하며, 탐지된 얼굴은 사용자가 선택적으로 편집할 수 있습니다.<br>
이를 통해 전문적인 편집 기술이 없어도 누구나 쉽게 초상권을 보호할 수 있으며, 안전한 영상 콘텐츠 제작이 가능합니다.<br>
공공기관, 의료기관, 일반 사용자 등 다양한 분야에서 개인정보가 포함된 영상을 효율적으로 보호하고 활용할 수 있도록 돕는 것을 목표로 합니다.<br>

## 다운로드 및 실행하는 방법

#### 1. 깃허브 내용 다운로드

cmd를 열고 `git clone https://github.com/catability/ai-face-blur-web.git` 입력

그럼 해당 경로에서 ai-face-blur-web 폴더가 생깁니다.


#### 2. 파이썬 가상환경 생성

경로는 크게 상관 없고, 굳이 가상환경을 만들지 않고 진행해도 되지만

그래도 로컬 환경에 최대한 영향 없도록 가상환경을 만들고 필요 라이브러리를 설치하여 진행하겠습니다.

경로 그대로에서 `python -m venv venv` 입력

그럼 venv 폴더 생깁니다.


#### 3. 가상환경 활성화

```
cd venv
cd Scripts
activate
```

그럼 가상환경이 활성화됩니다.


#### 4. 필요한 라이브러리 다운로드

```
cd ../..
cd ai-face-blur-web
cd backend
pip install -r requirements.txt
```

그럼 필요한 파이썬 라이브러리들이 설치됩니다.

#### 5. 웹 서버 실행

라이브러리 설치가 완료되면 `python main.py` 입력

잠시 기다리면 웹 서버가 실행되고, 발생하는 콘솔 출력 확인

웹 브라우저에서 웹 서버 경로(아마 127.0.0.1:5000)로 접속
