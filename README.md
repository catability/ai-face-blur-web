작품명: AI 기반 영상 얼굴 자동 블러 처리 웹서비스 구현

작품설명: <br>
  저희 작품은 AI 딥러닝 기술(YOLO)을 활용해 영상 속 얼굴을 자동으로 탐지하고 블러 처리하는 웹 서비스입니다.<br>
사용자가 영상을 업로드하면 AI가 실시간으로 얼굴을 인식해 익명화를 수행하며, 탐지된 얼굴은 사용자가 선택적으로 편집할 수 있습니다.<br>
이를 통해 전문적인 편집 기술이 없어도 누구나 쉽게 초상권을 보호할 수 있으며, 안전한 영상 콘텐츠 제작이 가능합니다.<br>
공공기관, 의료기관, 일반 사용자 등 다양한 분야에서 개인정보가 포함된 영상을 효율적으로 보호하고 활용할 수 있도록 돕는 것을 목표로 합니다.<br>

## 다운로드 및 실행하는 방법 (로컬에 직접 환경 구축과 도커를 이용한 컨테이너 실행 두 가지 방법 존재)

### 방법 1) 직접 환경 구축

#### 1. 깃허브 내용 다운로드

cmd를 열고 `git clone https://github.com/catability/ai-face-blur-web.git` 입력

그럼 해당 경로에서 ai-face-blur-web 폴더가 생깁니다.


#### 2. 파이썬 가상환경 생성

경로는 크게 상관 없고, 굳이 가상환경을 만들지 않고 진행해도 되지만

그래도 로컬 환경에 최대한 영향 없도록 가상환경을 만들고 필요 라이브러리를 설치하여 진행하겠습니다.

경로 그대로에서 `python -m venv venv` 입력

그럼 venv 폴더 생깁니다.


#### 3. 가상환경 활성화

```
cd venv
cd Scripts
activate
```

그럼 가상환경이 활성화됩니다.


#### 4. 필요한 라이브러리 다운로드

```
cd ../..
cd ai-face-blur-web
cd backend
pip install -r requirements.txt
```

그럼 필요한 파이썬 라이브러리들이 설치됩니다.

#### 5. 웹 서버 실행

라이브러리 설치가 완료되면 `python main.py` 입력

잠시 기다리면 웹 서버가 실행되고, 발생하는 콘솔 출력 확인

웹 브라우저에서 웹 서버 경로(아마 127.0.0.1:5000)로 접속

#### 6. 샘플 영상 업로드 및 결과 확인

ai-face-blur-web 폴더에는 samples 폴더가 있고, 그 안에 sample1.mp4 동영상이 있습니다.

해당 영상을 선택하여 작업을 진행해보세요.

<br>

### 방법 2) 도커를 이용한 실행
참고: 도커 이미지가 너무 커져서 AI 관련 GPU 관련 라이브러리들은 CPU 버전으로 대체해서 성능이 좀 떨어짐

#### 1. 도커 컨테이너 실행 (도커가 우선 설치되어 있어야 함.)

파워쉘을 열고 아래의 명령어를 입력합니다.(경로는 상관없음)

`docker run -d -p 5000:5000 --name ai-face-blur-web cattiy/ai-face-blur-web:v4`

현재 깃허브의 작업 환경을 그대로 도커 이미지로 빌드해놨고, 명령어를 통해 컨테이너로 실행합니다.

#### 2. 웹 브라우저 접속

(컨테이너 실행이 완료되면)웹 브라우저 아무거나 열고 `127.0.0.1:5000` 접속합니다.

#### 3. 샘플 영상을 다운로드 받고 테스트 해보기

`docker cp ai-face-blur-web:/app/samples .`

명령어를 실행한 경로에 samples 폴더가 생기고 그 안에 샘플 영상이 담겨있습니다.

해당 영상을 샘플로 사용해보세요.
